# Yii2-Admin
基于Yii2 Advanced开发的RBAC权限控制系统  

### 在线体验
[Yii2-Admin](http://106.75.117.140)

### 功能
- 后台管理员的登录/修改信息/退出/找回密码
- 权限管理
- 角色管理
- 管理员管理
